Organize receitas, despesas, contas, cartões e vencimentos em um só lugar.

## Baixar para Android

[**Baixar o APK do Financeiro Familiar**](https://github.com/syg-pedro/Planilha/releases/download/v1.0.0/Financeiro-Familiar-v1.0.0.apk)

> O aplicativo é instalado diretamente pelo APK; ele não está na Play Store.

1. Abra o link acima no celular Android.
2. Baixe e abra o arquivo APK.
3. Se solicitado, permita a instalação pela fonte usada (Chrome, WhatsApp ou Arquivos).
4. Abra o aplicativo, permita as notificações e entre ou crie sua conta.

## Atualizações

As melhorias normais chegam automaticamente. Basta deixar o app em segundo plano e abri-lo novamente.

Caso uma atualização Android seja necessária, um novo APK será publicado nas [Releases](https://github.com/syg-pedro/Planilha/releases) para instalar por cima da versão atual.
